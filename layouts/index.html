About NRCHKB

This is a collection of nodes which can be used to simulate HomeKit accessorys inside of Node-RED. Messages coming into these nodes are able to set accessory states and status in Apple's iOS and MacOS Home apps. Commands from Home.app (or Siri) will be passed from these nodes into your Node-RED flows.

Easy Install

Installation of this collection of nodes (node-red-contrib-homekit-bridged) is done directly in Node-RED from Manage palette.

Docker Install

You can also pull a Docker image containing everything needed to get started.

Examples Included

Example flows are included with the plugin, they can be imported using the Node-RED menu after install.
